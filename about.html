About This Project

Developed as part of my college project for S M Shetty College

Project Overview

This project is a Quiz Website developed as part of my college curriculum. It is designed to provide an interactive platform for users to take quizzes on various topics, track their scores, and improve their knowledge. The project is entirely developed by me, Dhyan Shah, as a demonstration of my skills in web development and database management.

The website includes features like user authentication, quiz selection, real-time scoring, and a user-friendly interface. It is built using HTML, CSS, JavaScript, PHP, and MySQL.

Key Features

User Authentication

Users can register, log in, and manage their profiles securely.

Quiz Selection

Choose from a variety of quizzes based on language and difficulty.

Real-Time Scoring

Get instant feedback on your quiz performance.

Responsive Design

The website is fully responsive and works on all devices.

Admin Panel

Admins can manage questions, users, and quiz results.

Database Integration

Uses MySQL to store and manage user data and quiz questions.
